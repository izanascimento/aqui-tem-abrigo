# Aqui Tem Abrigo

## Ideia do projeto

Minha ideia inicial foi criar uma solução simples para ajudar pessoas a encontrarem locais de abrigo de forma mais rápida e organizada em situações de emergência.

Na página inicial (home), os abrigos são apresentados com:
- Nome do local
- Endereço
- Disponibilidade de vagas
- Status de lotação

Os abrigos são classificados automaticamente como:
- Disponível
- Lotado

Também foi criado um botão de direcionamento para uma página individual do abrigo, onde o usuário consegue visualizar mais informações e acessar a opção de contato.

---

# Problema escolhido

## Falta de informação sobre abrigos e lotações

Em situações de emergência, muitas pessoas acabam se deslocando até locais que já estão lotados, enquanto outros ainda possuem vagas disponíveis.

A ausência de informações atualizadas dificulta o acesso rápido a lugares seguros e aumenta o desgaste das famílias afetadas.

---

# Solução proposta

A solução proposta foi desenvolver um sistema simples de visualização de abrigos, organizando:
- Capacidade total
- Quantidade de pessoas ocupando o local
- Status automático de lotação

O sistema foi pensado para ser simples e intuitivo, considerando diferentes tipos de usuários e priorizando facilidade de leitura.

Também utilizei diferenciação por cores:
- Verde para locais disponíveis
- Vermelho para locais lotados

para melhorar a identificação visual das informações.

---

# Estrutura da solução

O projeto foi desenvolvido utilizando:
- HTML
- CSS
- JavaScript
- VS Code

A estrutura conta com:
- Página inicial com listagem dinâmica de abrigos
- Renderização automática utilizando JavaScript
- Sistema automático de lotação
- Navegação entre páginas
- Página individual para cada abrigo

---

# Funcionalidades implementadas

- Exibição dinâmica dos abrigos
- Sistema automático de status
- Navegação para página individual
- Organização visual dos cards
- Sistema de cores para disponibilidade
- Estrutura responsiva básica

# Observações

Durante o desenvolvimento, tentei implementar imagens para cada abrigo, porém houve problemas no carregamento dos arquivos durante a configuração das pastas e caminhos das imagens.

Mesmo sem a funcionalidade concluída, a estrutura para imagens foi iniciada no projeto e permanece parcialmente implementada.
